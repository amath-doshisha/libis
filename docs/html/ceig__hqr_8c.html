詳解

多倍長精度実数型cmultiによるHessenberg型行列に前処理をした後にQR法で固有値を計算.

cmulti型の関数に関する定義はcmulti.cを参照のこと. cmulti型のベクトルの関数に関する定義はcvec.cを参照のこと. cmulti型の行列の関数に関する定義はcmat.cを参照のこと. cmulti型の固有値問題に関する定義はceig.cを参照のこと.
