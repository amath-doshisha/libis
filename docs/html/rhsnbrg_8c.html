詳解

多倍長精度実数型rmultiによるヘッセンバーグ型行列に関する関数の定義.

rmulti型の関数に関する定義はrmulti.cを参照のこと. rmulti型のベクトルの関数に関する定義はrvec.cを参照のこと. rmulti型の行列の関数に関する定義はrmat.cを参照のこと.
