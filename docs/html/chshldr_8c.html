詳解

多倍長精度実数型cmultiによるハウスホルダー変換に関する関数の定義.

cmulti型の関数に関する定義はcmulti.cを参照のこと. cmulti型のベクトルの関数に関する定義はcvec.cを参照のこと. cmulti型の行列の関数に関する定義はcmat.cを参照のこと.

chshldr.c に定義があります。
